# Betfair Clone

A full-stack sports betting application clone supporting real-time live betting.

## Structure

- `/frontend`: Next.js frontend application.
- `/backend`: Node.js/Express backend with Socket.io for real-time live odds updates.

## Getting Started

### Backend
1. Navigate to the `backend` directory.
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev` (or `node server.js`)

### Frontend
1. Navigate to the `frontend` directory.
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
